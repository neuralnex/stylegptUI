# StyleGPT - AI Fashion Assistant

StyleGPT is your personal AI-powered fashion assistant that helps you look your best every day. It simplifies outfit selection, wardrobe organization, and style discovery using intelligent recommendations tailored to your mood, occasion, and personality.

## Features

- Smart Outfit Suggestions
- Wardrobe Management
- AI-Powered Image Classification
- Fashion Chat & Wardrobe Chat
- User Authentication

## Tech Stack

- React
- TypeScript
- Vite
- SCSS
